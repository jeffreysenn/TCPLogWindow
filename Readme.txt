Project member: Xiaoyue Chen
Project name: TCP Log Window
Introduction: 
The project contains a server and a client program. User can choose to launch either of them
by changing commands. 
Server: user can specify a port for the listener. The server can accept multiple clients and 
reconstruct and process clients' requests. The server also logs and replies to the clients. 
Client: user can specify the server's ip address and connect to the server. The client can 
send requests to the server. User can input the request body. It also receives the response 
from the server. It logs every request it sents and every response it receives from the server.
